## O que é este projeto?
Este trabalho é sobre a "Arte da Abstração". Basicamente, pegamos uma situação do dia a dia (fazer uma compra na internet) e limpamos toda a bagunça dela até sobrar apenas o que é importante para um computador entender.

## Como o trabalho foi dividido:

1. **O mundo real:** Aquela lista enorme de tudo o que a gente faz (abrir o site, clicar no produto, olhar o preço).
2. **O meio do caminho:** Um desenho simples (fluxograma) que mostra só as decisões principais, tipo "tem estoque?" ou "o cartão passou?".
3. **A lógica final:** O passo a passo organizado que está no arquivo `pseudocodigo.md`.

## Por que fizemos assim?
Usamos o pensamento computacional para:
* Não se perder nos detalhes (Abstração).
* Ver o que sempre se repete em qualquer compra (Padrões).
* Criar uma solução que serve para qualquer loja (Generalização).
